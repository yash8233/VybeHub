## How to Add Screenshots

1. Take clear screenshots of your VIBE application
2. Export them as PNG files
3. Name them according to the above naming convention
4. Place them in this `docs/` folder
5. The README.md will reference these screenshots automatically

## Recommended Screenshot Dimensions

- **Width:** 1280px or higher
- **Format:** PNG
- **Quality:** High quality/uncompressed
